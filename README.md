# City Master #

A webbrowser multiplayer city building game.
This game uses CartoDB to render the city map.

### Requirements ###

* PHP 5.5
* MySQL
* Webserver with rewrite mod
* CartoDB account

### Setup ###

* Edit /conf/config.php
* Import /conf/db.sql & delete the file
* Enjoy
